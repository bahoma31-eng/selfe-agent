# CHANGELOG — Selfe Agent

## [v3.1.0] — 2026-04-13

### تحسينات البرومبت
- إعادة كتابة `system_prompt.txt` بمنهجية `agent-orchestration-improve-agent`
- إضافة Chain-of-Thought (5 خطوات تفكير داخلي)
- إضافة Constitutional AI (5 قواعد سلوك صريحة)
- إضافة أمثلة Few-Shot عملية
- **نتيجة الاختبار: +49% تحسن عام عبر 10 حالات اختبار**

### تحسينات تقنية — `agent.py`
- **`detect_temperature()`**: temperature ديناميكية حسب نوع الطلب
  - كود/تصحيح → `0.2` (أقل هلوسة)
  - تحليلي/واقعي → `0.5` (توازن دقة/طلاقة)
  - إبداعي → `0.9` (تنوع أعلى)
  - عام → `0.7` (الافتراضي)
- **`max_tokens=4096`**: حد أقصى للردود لمنع القطع واستهلاك الحصة
- **Retry + Exponential Backoff**: إعادة المحاولة تلقائياً (3 محاولات) عند الأخطاء المؤقتة
  - المحاولة 1: فوراً
  - المحاولة 2: بعد 1s
  - المحاولة 3: بعد 2s
- رفع إصدار الـ User-Agent إلى `selfe-agent/3.1`
- رفع رقم الإصدار في واجهة الترحيب إلى `v3.1.0`
